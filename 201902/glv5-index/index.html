这里是盖伦特5级任务索引，推荐任务将以列表形式出现在这篇博客中。任务攻略将先以文档的形式发布，视频通过审核后延时发布，请收藏本网页并随时回来检查更新。

对于高安党而言5级任务比较神秘。

  • 一方面是可能是觉得5级比4级还高级，应该难度很大；
  • 另一方面，5级任务的代理人都在低安，任务地点可能会指引您到一些0.0的区域（不推荐）。

但是当您了解了5级任务的任务流程以后，往往会惊呼！这么简单的吗？这篇5级任务攻略系列将带您揭开5级任务的神秘面纱，当您掌握了这种方法，1个小时收入5亿绝对不是难事。

4. 任务流程

要想短时间获取最大的利润，您最好拥有一个旗舰账号（carrier pilot），并配上多个能够接5级任务的账号（puller）：

  1. 用小号创建一个舰队，组上你的旗舰号（小队指挥）；
  2. 按照以上推荐接任务，如果是小号接的任务就小队跃迁到任务地点100km，否则旗舰号自己单独跃迁；
  3. 确认旗舰号进入跃迁状态，小号按停止跃迁并返回任务代理人空间站准备交任务；
  4. 旗舰号完成任务目标，小号交任务再刷新任务；
  5. 重复上述流程。

注意: 当没有接到上述任务时，每4个小时可以拒绝一次不掉声望；如果您的势力声望比较高(>7.0)，就可以持续拒绝，但是要注意代理人的声望不要低于-2.0，否则代理人会拒绝刷新任务。

Tips:

  • 你可以在任务星座的每个星系都放上一艘任务船，平时跑路使用截击、阿斯特罗甚至是穿梭机，可以节省跑路时间;
  • 使用0m进站（自己出站的时候做一个位标）。由于模型的原因，使用默认的跃迁0m到空间站可能实际上还要1000m，这个时候您需要慢慢的爬到0m才能被牵引到站内。
