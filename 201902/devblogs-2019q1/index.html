劲爆消息！EVEOnline免费账号将无法接受4级及以上任务

Games EVEOnline Missions 2019/02/21

原文链接 SECURITY UPDATE - Q1 2019

2019.2.19日，CCP像平常一样公布了每季度的安全报告，一些常规的封号公示毫无新意，但一项劲爆的消息给目前免费的游戏机制带来重大改变：Alpha账号（免费账号）将无法接受4级及以上任务的委托！

背景

2003年EVEOnline正式推出以来，一直都是老玩家链接提供新玩家21天免费试玩的邀请机制。10多年来，EVEOnline由于各种原因，玩家数量一直未大幅提升，为了吸引更多的玩家进入这座新伊甸园，2016年底，CCP竟然宣布游戏免费了！

帅气的元帅级隐秘战列舰

免费的游戏必然会带来新人，但是游戏本身机制比较硬核，开放性和玩法虽然跨越那个时代，但在今天看来，如果不作出较大改变，EVEOline还将继续不愠不火的运行下去。

  • eve奖励机制非常平淡，而惩罚对于新人来说太高了；
  • 刷钱、挖矿的操作近乎机械式的重复劳动，毫无乐趣而言；
  • 制造、运输、管理库存、关注市场、矿产收购和商品的卖出有时会让人感觉非常枯燥；
  • PVPPPP的模式让单打独斗无法发挥，断了氪金党开启无双模式的念头。
  • ……

总之，虽然整个游戏都是玩家生产的内容，但游戏本身机制太硬核，如果找不到很好的方法，获取与付出不成比例，也是为什么那么多新人尝试了留也不下的原因。

由于EVEOnline操作重复程度较高，非常适合编写相应的程序代替人工完成枯燥的工作，因此大量脚本出现在这个游戏中，也毁了EVE China。

免费没有引入更多的新人，反而可能扩大化脚本的使用，因为免费账号可以毫无顾忌的使用脚本，被封号之后大不了再申请一个新的，所以CCP在打开一扇门之后，就要想办法关闭一扇门，与脚本和RMT的战斗从未停止。

事件影响

正因为CCP发现已经有脚本能够自动完成4级和5级快递任务，CCP调研后起初认为只需关闭快递任务即可，然而在经过周密的调查后竟然发现大多数4级和5级任务都有被免费账号滥用的情况。

目前，该改动已经在测试服务器上生效，今后Omega账号才能够进行高等级任务很有可能会成为现实。至于该重大改动会有什么影响，请大家拭目以待！至少我认为有：

  • 脚本还将继续存在，目前的Plex价格相对作案成本来说太低；
  • Plex销量会增大并将继续涨价；
  • 大量高安玩家可能会尝试进入0.0军团，VNI图纸会涨价；

Bot被封继续注册一个，只是现在需要付费了增加了一点成本而已，我想这个最大的受益者应该是CCP。而对玩家来说，自从CCP被黑沙收购以后就没做几件好事，各种活动毫无诚意，Plex从2018年6月的2.7m飙升至目前的3.7m，现在还在持续上涨，情人节竟然象征性的-10%，我也就呵呵了。

我只想吐槽：“游戏机制就很棒子了，才会被棒子看中”，最后奉劝一句不作死就不会死！与其费尽心思通过修改游戏机制扩大销售额，不如想想怎么改动游戏机制，真正的造福于广大玩家。
