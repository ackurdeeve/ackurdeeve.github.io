刀塔自走棋英雄列表工具数据更新（2.27号数据）

Dota2 Dota2 对战模拟 查询工具 自走棋模拟 阵容模拟 2019/03/01

刀塔自走棋2月27号进行了小幅更新，除了修改疯脸和刃甲的属性，还包括英雄基础攻击数据的修改。

物品更新

  • 疯狂面具的效果由吸血+10%、攻击速度+30、-10护甲调整为吸血+10%、攻击速度+40、并沉默自己
  • 刃甲的被动伤害反弹效果由10%提高为20%

道具目前还没有上线查询工具，本次更新仅修改了英雄数据：

棋子更新

  • 隐形刺客的烟雾时间从 3 / 4 / 5秒延长至 4 / 5 / 6秒
  • 炼金术士的攻击力从 50-70 / 100-140 / 200-280提高至 50-80 / 100-160 / 200-320
  • 死亡先知的攻击力从 40-50 / 80-100 / 160-200 提高至 50-60 / 100-120 / 200-240

目前英雄的小幅改动并未影响阵容的配置，但疯脸的改动给不少玩家造成不小的困扰，主要是附加沉默属性，无法使用技能，建议给装备被动技能的英雄使用。

  • 幻影刺客
  • 卓尔游侠
  • 敌法师
  • 月之骑士
  • 巨魔战将
